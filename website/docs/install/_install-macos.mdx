<summary>PATH</summary>

In order to access the OpenOCD binaries installed in
the central xPacks store,
the project build configurations need a PATH adjustment:

<CodeBlock language="sh"> {
`export PATH=$HOME/Library/xPacks/\@pack-dev-tools/openocd/${ customField('upstreamVersion') }-${ customField('xpackSubversion') }.${ customField('npmSubversion') }/.content/bin:$PATH`
} </CodeBlock>
</details>

} </CodeBlock>
