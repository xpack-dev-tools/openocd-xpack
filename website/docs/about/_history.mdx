<HistoryGnuArmEclipse date="August 2015"/>

The initial name of the project was
[GNU ARM Eclipse OpenOCD](https://github.com/ilg-archived/openocd)
and was first published on `npmjs.com` as
[`@gnu-mcu-eclipse/openocd`](https://www.npmjs.com/package/@gnu-mcu-eclipse/openocd).

### January 2015

The first **GNU ARM Eclipse OpenOCD** release was
[v0.8.0-20150119](/blog/2015/01/19/openocd-v0.8.0-20150119-released/).

The next few releases provided only `*-setup.exe` binaries for x86 Windows.
