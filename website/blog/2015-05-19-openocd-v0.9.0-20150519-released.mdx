Version **0.9.0-20150519*** is a new release; the just released stable version of OpenOCD (**0.9.0**) is now also available as ready to install, platform specific, binary files.

## Download

The binary files are available from <a href={frontMatter.download_url}>GitHub Releases</a>.

## Notes

Install the proper package, and then update the path in _Eclipse_ menu →  **(Window →) Preferences** → **Run/Debug** → **OpenOCD** to the new folder. If installed in the default location, the **Restore Defaults** button should automatically configure the proper path.
